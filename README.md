# Kellendonk.ca

Repository containing all the Kellendonk.ca stuff.

## Contributing

**Editing the Website**

```sh
# Install dependencies
yarn install
# Enter watch mode for the nextjs app
yarn nx serve web
```

**CDK App**

```sh
# Install dependencies
yarn install
# Enter CDK watch mode for the dev stage
yarn nx serve cdk
```
